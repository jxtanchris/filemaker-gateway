# Console 深色主题视觉升级 — 设计规格

日期: 2026-07-23
状态: 已确认
类型: 视觉升级 (Visual Enhancement)

## 概述

将 FileMaker AI Gateway 测试控制台 (`console.html`) 从简约浅色风格升级为深色科技感风格，聚焦纯视觉改进，不动 HTML 结构和 JS 逻辑。

## 设计原则

- **纯视觉升级**: 只动 CSS / 少量 HTML class 微调，不改 JS 逻辑
- **深色科技风**: 深蓝黑底 + FileMaker 品牌青蓝色调
- **玻璃拟态**: 侧边栏和弹窗使用毛玻璃半透明效果
- **克制动画**: 过渡流畅但不喧宾夺主

## 组件设计

### 侧边栏

- 玻璃拟态: `background: rgba(15,20,35,0.88); backdrop-filter: blur(20px);`
- 右边框: `1px solid rgba(255,255,255,0.06)`
- Logo 区: 图标 + 标题 + 版本号，标题使用青色 `text-shadow` 微发光
- 状态指示器: 绿色圆点 + `pulse` 动画（scale 1→1.3→1, 2s 循环）
- 工具列表: `已注册工具` section title 小写加大字距
- 工具项: 圆角 8px, hover 时 `rgba(6,182,212,0.08)` 背景亮起, 0.2s 过渡
- live badge: 改为绿色小圆点 `●`，不用文字
- 自定义滚动条: 6px 宽，`rgba(255,255,255,0.1)` 滑块

### 顶部栏

- 背景同主背景色，底边框分割
- 会话输入框: 背景 `#0f172a`，圆角 8px，聚焦时青色边框 + `box-shadow` 光晕
- 按钮: 边框 `rgba(255,255,255,0.1)`，hover 时边框变青，0.2s

### 聊天区

- 用户气泡: 青色渐变背景，白色文字，右对齐
- AI 气泡: `#1a2332` 背景，`rgba(6,182,212,0.15)` 边框，左对齐
- 代码块: `#0c1220` 背景，JetBrains Mono，顶部语言标签
- 表格: 暗色表头 `rgba(6,182,212,0.1)`，hover 行微亮
- 思考动画: 三点跳动（bouncing dots），0.4s 交错延迟

### 输入区

- 输入框: `#0f172a` 背景，内凹阴影 `inset 0 2px 4px rgba(0,0,0,0.3)`
- 聚焦: 青色 `box-shadow` 光晕扩散
- 发送按钮: 青色渐变，hover 亮度提升 10%
- 图片按钮: 圆形，hover 旋转 15°

### 会话弹窗

- 遮罩: `rgba(0,0,0,0.6)`
- 弹窗: 玻璃拟态，与侧边栏统一
- 打开动画: scale 0.95→1.0 + fade in, 0.2s
- 关闭按钮: hover 旋转 90°, 0.3s
- 列表项: hover 左边出现青色 3px 边框

## 动画清单

| 元素 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 指示灯 | pulse 脉冲 | 2s infinite | ease-in-out |
| 消息出现 | fadeIn + slideUp (8px) | 0.3s | ease-out |
| 思考指示器 | 三点依次弹跳 | 0.4s 交错 | ease |
| 按钮/工具 hover | 背景/边框颜色过渡 | 0.2s | ease |
| 输入框聚焦 | 光晕扩展 | 0.3s | ease |
| 弹窗 | scale + fade | 0.2s | ease-out |
| 图片按钮 hover | rotate 15deg | 0.3s | ease |
| 关闭按钮 hover | rotate 90deg | 0.3s | ease |

## 其他细节

- 选中文字高亮: `rgba(6,182,212,0.3)`
- 链接色: `#06b6d4` + 下划线
- 图片预览 chip: 暗色圆角标签
- 空状态: 六边形 Logo `⬡` + 标题 + 提示文字

## 范围边界

**包含:**
- 重写 `<style>` 块所有 CSS
- 微调 HTML 中部分元素的 class/结构（如语言标签、空状态）
- 引入 Google Fonts

**不包含:**
- JS 逻辑改动
- 新功能添加
- 流式输出
- 移动端适配
- 后端改动
